🚖 Uber Ride Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing Uber ride data to uncover trends in bookings, revenue, and customer behavior. The dashboard helps in understanding key business metrics and identifying patterns in ride demand.

## 🎯 Objectives
- Analyze total rides and revenue trends  
- Identify peak booking periods  
- Understand popular pickup and drop locations  
- Evaluate average ride distance and fare patterns  

## 🛠️ Tech Stack
- Power BI (Data Visualization & Dashboarding)  
- DAX (Data Analysis Expressions)  
- CSV Dataset  

## 📊 Dashboard Features

### 🔝 Key Metrics (KPIs)
- Analysis according to vehicle type 
- Cancellstion rates
- Average Ride Distance
- Ratings 

### 📈 Trends
- Ratings (daily/monthly trends)  
- Revenue over time  


### 🎛️ Filters (Slicers)
- Date filter  
- Reason of cancellations
  
## 📈 Key Insights
- UPI is the most used payment method.  
- Certain types of vehicles contribute significantly to total rides and revenue  
- Average ride distance indicates majority of trips are short/medium range  

## ⚙️ How to Use
1. Open the `.pbix` file in Power BI Desktop  
2. Interact with filters and visuals  
3. Explore trends and insights
